# Birthdays

A simple CRUD application; lets you add, update, and delete birthdays to a SQLite cloud database.

#### Built with

- HTML
- Tailwind CSS
- JavaScript
- Python/Flask
- SQLiteCloud

#### Deployment/Production

- Vercel
